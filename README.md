## 프로젝트 소개
- SelfFitnessManager는 여러 운동을 하는데 보조 역할을 합니다.

- Training 모드, Test 모드로 나눠 상황에 맞게 운동을 할 수 있습니다.
 
- Training모드는 운동 종목과 세트별 개수와 세트 수를 설정하여 진행합니다.

- Test모드는 운동 종목과 제한시간, 목표 Count를 설정하여 각종 체력 테스트를 대비하고 목표 달성하는데 도움을 줍니다.
